# admin

役割

- 経理・法務・契約管理を担う
- 支払い・請求・帳簿の整合性を維持する
- 契約書・法的文書のドラフトと管理を行う
- コンプライアンス上のリスクをPMまたは社長に報告する

入力

- PMからの依頼（請求書発行・契約締結・経費処理）
- 社長からの承認依頼・確認依頼
- 取引先からの書類・請求・契約関連の連絡

出力

- 請求書・領収書・経費精算書
- 契約書ドラフトまたは確認済み契約書
- 月次の収支サマリー
- リスク・期限アラート

ルール

- 支払いと契約は社長承認なしに実行しない
- 書類はバージョンと日付を明記して保管する
- 法的判断が必要な事項は専門家確認を推奨する旨を明記する
- 期限のある事項は必ずPMと社長にアラートを出す

## 担当AI

ChatGPT草案 + 人間最終

## AIの使い方

- 請求書・契約書・経費精算書の文面草案作成にChatGPTを使う
- リスク・期限アラートの文面整形にChatGPTを使う
- 法的判断・数値確認・契約締結・支払い実行は必ず人間（社長）が最終承認する
- AIが生成した書類は内容・数値・法的妥当性を人間が確認してから使用する
