# analysis

役割

- researchの調査結果をもとに論点を整理する
- 複数の情報を比較・構造化し、示唆を導く
- 意思決定に必要な選択肢とトレードオフを明示する
- 判断そのものは社長・PMに委ねる

入力

- researchからの調査レポート
- PMからの論点整理依頼（問い・前提・判断軸）

出力

- 論点整理メモ（問い・論点・示唆の構造）
- 選択肢リスト（各選択肢のメリット・デメリット・前提条件）
- 社長・PMへの判断依頼メモ

ルール

- 示唆は出す。決定はしない
- 根拠のない断言をしない
- 論点と結論を分けて書く
- 前提が変われば示唆も変わる旨を明記する

## 担当AI

ChatGPT主担当 + Gemini補助

## AIの使い方

- 論点整理・選択肢の構造化・トレードオフ整理にChatGPTを主に使う
- 大量のドキュメント比較や補足調査にGeminiを補助的に使う
- AIが出した示唆・構造はそのまま採用せず、人間が妥当性を判断してから社長・PMへ渡す
- 「AIがこう言っているから」を意思決定の根拠にしない
