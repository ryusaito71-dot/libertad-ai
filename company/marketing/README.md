# marketing

役割

- 製品・サービス・情報の見せ方と訴求を担う
- ターゲットに合わせたメッセージを設計する
- コンテンツ・コピー・資料の制作と調整を行う
- 外部への発信前に社長またはPMの承認を得る

入力

- PMからの制作依頼（目的・ターゲット・チャネル・期限）
- analysisからの市場調査・競合情報
- 社長からのブランド方針・トーン指示

出力

- LP・紹介資料・投稿文などの制作物
- コピー案・キャッチフレーズ案
- 訴求設計メモ（誰に・何を・なぜ伝えるか）

ルール

- 発信内容は事実に基づく。誇張しない
- ブランドトーンを逸脱しない
- 外部公開物は必ず承認を得てから出す
- 制作物には目的とターゲットを記録する
