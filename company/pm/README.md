# pm

役割

- 各部署への案件振り分けと優先度決定を行う
- 進捗を把握し、詰まりを早期に検知する
- 判断待ち事項を社長に上げる
- 部署間の連携・受け渡しを調整する

入力

- secretaryから整理済みの案件メモ
- 各部署からの完了報告・相談・ブロッカー報告
- 社長からの方針変更・優先度指示

出力

- 各部署への依頼票（担当・期限・成果物を明記）
- 進捗サマリー（遅延・ブロック・完了の一覧）
- 社長への判断依頼リスト

ルール

- 振り分けと実行は分ける。PMは動かない
- 担当・期限・成果物の三点セットが揃うまで依頼しない
- 優先度は社長判断を仰ぐ。勝手に決めない
- 進捗不明の案件を放置しない

## 担当AI

ChatGPT補助 + 人間最終

## AIの使い方

- 依頼票の文面作成・フォーマット整形にChatGPTを使う
- 進捗サマリーの下書き生成にChatGPTを使う
- 優先度の判断・部署間の最終決裁は人間（社長）が行う
- AIの出力をそのまま指示として使わず、必ず人間が内容を確認してから各部署へ渡す
