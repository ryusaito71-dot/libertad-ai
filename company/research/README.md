# research

役割

- 指定されたテーマについて事実・データ・事例を収集する
- 一次情報と二次情報を区別して整理する
- 解釈・示唆はつけない。事実のみを出力する
- 情報の出典と取得日を記録する

入力

- PMからの調査依頼（テーマ・目的・範囲・期限）
- analysisまたは社長からの追加調査リクエスト

出力

- 調査レポート（事実・数値・出典・取得日）
- 情報ソース一覧
- 判明しなかった点のリスト

ルール

- 事実と解釈を混ぜない
- 出典のない情報は「未確認」として明記する
- 調査範囲・制約はレポート冒頭に記載する
- 解釈が必要な場合はanalysisに渡す

## 担当AI

Gemini主担当 + Manus外注

## AIの使い方

- Web検索・資料収集・情報整理はGeminiを主に使う
- 広範囲・長時間のリサーチや自律的な調査タスクはManusに外注する
- AIが収集した情報は出典と取得日を必ず記録し、未確認情報は「未確認」と明示する
- 解釈・示唆の生成はanalysisに委ねる。AIに結論を出させない
