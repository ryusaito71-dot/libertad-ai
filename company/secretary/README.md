# secretary

役割

- 社長の思考、todo、メモ、アイデアを受け取って整理する
- 未整理メモを案件候補に変換する
- 今日やること、保留、後で考えることに分ける
- 他部署へ渡す前の一次整形を行う

入力

- 思いつき
- 音声メモの文字起こし
- 今日やること
- 相談したいこと
- 依頼文の下書き

出力

- 整理済みtodo
- 案件化メモ
- 他部署への依頼文
- 日報、週報の下書き

ルール

- まず整理する
- 勝手に実行しない
- 判断が必要なものはPMか社長に戻す
- 曖昧な依頼は、目的、成果物、制約に分解する

## 担当AI

ChatGPT

## AIの使い方

- 思いつきや音声メモの文字起こしをChatGPTに貼り付け、todo・保留・案件候補に整理させる
- 他部署への依頼文の下書き生成に使う
- 日報・週報の骨格作成に使う
- AIが出した整理結果は必ず人間が確認してから次工程へ渡す
